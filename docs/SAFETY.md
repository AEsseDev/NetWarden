# Безопасность

## Защищенные процессы

NetWarden считает защищенными процессы, связанные с Riot/League/Vanguard. Они:

- помечаются в UI как `ЗАЩИЩЕНО`;
- не обрабатываются watchdog для `Пауза/Завершить`;
- не могут быть переведены в destructive сценарий через правила.

## Ограничения

- Приложение работает без `sudo` и не управляет root-only демонами.
- Метрики сети зависят от доступности и точности `nettop`.

## Рекомендации

- Не добавлять в правила процессы, связанные с самой игрой и античитом.
- Проверять логи перед изменением агрессивности правил.
